# OIBSIP Data Analytics Internship – Task 4

## Task: Credit Card Fraud Detection

### Objective
Detect fraudulent credit card transactions using machine learning techniques to help identify and prevent financial fraud.

### Steps Performed
- Loaded and explored the credit card transactions dataset
- Handled class imbalance between fraudulent and genuine transactions
- Performed data preprocessing and feature scaling
- Built classification models to detect fraudulent transactions
- Evaluated model performance using Precision, Recall, F1-Score, and ROC-AUC
- Visualized the confusion matrix and ROC curve

### Tools Used
Python, Pandas, Scikit-learn, Logistic Regression, Random Forest, Matplotlib, Seaborn, Jupyter Notebook

### Outcome
Built a classification model capable of identifying fraudulent credit card transactions, with performance evaluated using precision, recall, and ROC-AUC to balance fraud detection accuracy against false alarms.
